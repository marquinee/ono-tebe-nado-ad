# Оно тебе надо

Учебный проект по вёрстке одностраничного сайта аукциона вещей, в которые никто не верил.  
В проекте реализованы семантическая HTML-разметка, стилизация с помощью CSS, сетки `grid`, `flex`, фоновые изображения, карточки лотов, секция об аукционе и подвал с навигацией и социальными ссылками.
